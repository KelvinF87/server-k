Auth Routes

Sign Up

POST http://localhost:5005/auth/signup
Content-Type: application/json

{
  "username": "new_user",
  "email": "new_user@example.com",
  "password": "password123",
  "name": "Name",
  "lastname": "Lastname",
  "roles": ["user"],
  "active": true,
  "image": "url_to_image"
}

Login

POST http://localhost:5005/auth/login
Content-Type: application/json

{
  "username": "new_user",
  "password": "password123"
}

Verify Token

GET http://localhost:5005/auth/verify
Authorization: Bearer <tu_token_jwt>

Gasto Routes

Create a New Gasto

POST http://localhost:5005/api/gastos
Content-Type: application/json
Authorization: Bearer <tu_token_jwt>

{
  "gasto": 100,
  "tipo": "60b8d28e73c68e2a781a4b61",
  "detalles": "Compra de comida",
  "balance": 50
}

Get All Gastos

GET http://localhost:5005/api/gastos
Authorization: Bearer <tu_token_jwt>

Get a Gasto by ID

GET http://localhost:5005/api/gastos/:id
Authorization: Bearer <tu_token_jwt>

Update a Gasto

PUT http://localhost:5005/api/gastos/:id
Content-Type: application/json
Authorization: Bearer <tu_token_jwt>

{
  "gasto": 150,
  "detalles": "Compra de comida actualizada"
}

Delete a Gasto

DELETE http://localhost:5005/api/gastos/:id
Authorization: Bearer <tu_token_jwt>
